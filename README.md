# 🧮 Multiplication Game

A simple and fun multiplication quiz game built with React and Firebase Realtime Database.  
Players solve math questions under time pressure and compete for the highest score.

---

## 🚀 Features

- 🎮 Fast-paced multiplication gameplay
- ⏱️ Timer-based challenge (60 seconds)
- 🏆 Leaderboard system (Top 5 players)
- 💾 Real-time data storage with Firebase Realtime Database
- 👤 Player name input
- 📊 Dynamic score calculation based on difficulty and correct answers
- 🔄 Live leaderboard updates

---

## 🛠️ Tech Stack

- React (Vite)
- React Router
- Firebase Realtime Database
- JavaScript (ES6+)
- CSS
